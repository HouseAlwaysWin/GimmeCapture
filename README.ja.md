# GimmeCapture!! 🦊 (Japanese)

スタイリッシュで高性能なスクリーンキャプチャ＆録画ツール。名前の由来は BABYMETAL の名曲 **[「ギミチョコ！！」(Gimme chocolate!! Official Video)](https://www.youtube.com/watch?v=WIKqgE4BwAY)** です。

---

### 🎸 デザインのインスピレーション
**GimmeCapture!!** は、高速でエネルギッシュ、そして強烈なスタイルを目指しています。UI の配色やテーマは BABYMETAL の美学をオマージュしています。

### 主な機能
- **スマートキャプチャ**：インスタント編集ツールを備えた高性能スクリーンキャプチャ。
- **画面録画**：MP4、MKV、GIFなどの形式で画面を録画（FFmpegを使用）。
- **最前面にピン留め**：キャプチャをフローティングウィンドウとしてピン留め。
- **編集ツール**：キャプチャ上に直接、図形、矢印、線、テキストを描画。
- **カスタマイズ可能なキー**：「操作」タブで全てのショートカットキーを自由に設定可能。
- **個性化外観**：枠線の太さ、マスクの不透明度、テーマカラー（ゴールド、シルバー、レッド）を調整可能。
- **スタートアップ起動**：Windowsの起動時に自動的にアプリを開始可能。

### 使い方
1. **F1 (デフォルト)**：キャプチャ開始。
2. **F2 (デフォルト)**：録画の開始/停止。
3. **F3 (デフォルト)**：キャプチャしてピン留め。
4. **Ctrl + C**：クリップボードにコピー。
5. **右クリック**：ピン留めされたウィンドウや画面上でコンテキストメニューを表示。

### 📦 サードパーティコンポーネント
- **FFmpeg**: 画面録画とマルチメディア処理に使用されます。FFmpeg は [GPL/LGPL](https://ffmpeg.org/legal.html) ライセンスの下で提供されています。本アプリは、[BtbN/FFmpeg-Builds](https://github.com/BtbN/FFmpeg-Builds) から FFmpeg バイナリを自動的にダウンロードします。
